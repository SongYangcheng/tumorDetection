#### 安装教程

1. 后端配置: `pip install -r requirements.txt`
2. 前端配置: `cd frontend && npm install`
3. 运行后端: `python backend/main.py`
4. 运行前端: `cd frontend && npm run dev`

#### 使用说明

1. 访问前端界面
2. 上传医学图像
3. 查看检测结果
4. 下载报告
